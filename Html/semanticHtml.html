Understanding HTML Content Categories

  • Metadata
  • Flow
  • Sectioning
  • Heading
  • Phrasing
  • Embedded
  • Interactive

Metadata

Metadata content provides information about the HTML document, typically used within the <head> element.

Flow

Flow content includes the majority of content in an HTML document. It can contain headings, sections, and embedded content.

Sectioning

Sectioning content defines the structure of the document, like <header>, <footer>, <section>, <article>, and <aside>.

Heading

Heading content defines the headings of a section, typically using <h1> to <h6> elements.

Phrasing

Phrasing content defines the text and its structure within an element, including inline elements such as <a>, <span>, <em>, and <strong>.
